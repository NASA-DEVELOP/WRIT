==================
TXT File for Sentinel Alcolite
==================

Creates a txt file containing the link to all raster folders within the designated folder. The text file should be used as the
"Image=" in the batch ACOLITE operation. This tool will generate the TXT needed for the Landsat image list to batch ACOLITE.
This tool will generate the TXT needed for the Sentinel image list to batch ACOLITE.

 Required Packages
===================
* ESRI ArcMap

 Parameters
-------------

1. Navigate to the toolbox Water Resources Integration Toolbox, run TXT File for Sentinel Alcolite.
2. Input workspace as a folder where your image folders for Sentinel reside.
